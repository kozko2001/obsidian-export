
# Roam export 

This is a dirty cli to export my obsidian files stored in a nextcloud folder

## install dependencies

```
pip install -r requirements.txt
```

## Run it

set the correct env variables:

- `NEXT_CLOUD_HOST` (i.e: https://cloud.domain/com)
- `NEXT_CLOUD_USER`
- `NEXT_CLOUD_PASSWORD`
- `NEXT_CLOUD_FOLDER`

run 

`python main.py`
